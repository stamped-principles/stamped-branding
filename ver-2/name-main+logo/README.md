# Main Logo (v2)



## Inkscape instructions

Refer to the `_inkscape.json` parameters for precise details of how to manually reconstruct the logos.

Note that all **positions** (x,y) and **dimensions** (width, height) apply _after_ rotation (which alters the raw values following transformation).

Also note that the custom font used by produce strange behavior on platforms without the font installed; as such the object in Inkscape must be copied as a path (a backup of the text is stored in the hidden layer 2 for easy modification).
